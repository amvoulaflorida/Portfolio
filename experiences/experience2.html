Projet de Scan d'Ordonnance Médicale

Date : 2024

Société : SOCIÉTÉ MEDICUSCLINIC

Pays : Usa

Projet(en ligne): Ce projet permet de scanner une ordonnance médicale et restituer ce qui a été scanné sous un format numérique analysé et structuré.

Objectif

l’application permet de scanner des documents médicaux (ordonnances), d’extraire les informations clés (nom des médicaments, dosages, durée du traitement, etc.), puis de restituer ces données sous un format structuré pour les patients, médecins, ou pharmacies.

Problèmes

  • Les ordonnances sont souvent manuscrites, ce qui peut entraîner des erreurs de transcription.
  • La gestion des ordonnances papier est lente et nécessite beaucoup de ressources.
  • Il existe un besoin d'automatiser l'extraction des données pour améliorer l'efficacité.

Solution

Développement d'une application capable de scanner une ordonnance médicale, d'en extraire le texte via une analyse OCR et de restituer ces informations dans un format numérique analysé.

Besoins fonctionnels

  • Scanner l'image de l'ordonnance médicale.
  • Extraire les informations médicales pertinentes (médicaments, posologie, etc.).
  • Restituer ces informations sous un format structuré dans un fichier numérique ou une base de données.

Outils

Python

Django (framework )

OCR La reconnaissance optique de caractères

API Postman .

Visual studio code

Conclusion

Ce projet permet d'améliorer l'efficacité dans la gestion des ordonnances médicales en automatisant l'extraction et la numérisation des informations, facilitant ainsi l'accès et la gestion des données de santé.
